## 4. Adding Custom Google Fonts

The `<link>` tag is already included in the boilerplate above. Here's what it loads and why:

| Font | Role | Weights Used |
|------|------|--------------|
| **Inter** | Headings, UI, nav | 300, 400, 500, 600, 700, 800, 900 |
| **DM Sans** | Body text, captions | 300, 400, 500, 700 (+ italic 400) |

```html
<!-- Already in <head> 芒聙聰 do not duplicate -->
<link
  href="https://fonts.googleapis.com/css2?family=Inter:wght@300;400;500;600;700;800;900&family=DM+Sans:ital,opsz,wght@0,9..40,300;0,9..40,400;0,9..40,500;0,9..40,700;1,9..40,400&display=swap"
  rel="stylesheet"
/>
```

### CSS font-family declarations

These are set in your overrides file (see Section 5), but here's the pattern:

```css
/* Headings 芒聙聰 Inter */
h1, h2, h3, h4, h5, h6, .nav-logo, .section-heading {
  font-family: 'Inter', -apple-system, BlinkMacSystemFont, sans-serif;
}

/* Body 芒聙聰 DM Sans */
body, p, li, a, button, input, textarea {
  font-family: 'DM Sans', -apple-system, BlinkMacSystemFont, sans-serif;
}
```

> 冒聼聮隆 **Performance tip:** `display=swap` is already in the Google Fonts URL. This ensures text renders immediately in a fallback font while Inter/DM Sans loads, preventing invisible text flash (FOIT).

### OPTION A 芒聙聰 iframe Approach (Recommended for most cases)

```javascript
      doraIframe.contentWindow.postMessage({
        type: 'NEXEVO_BRAND_TOKENS',

          primaryColor:    '#B3FE85',
          backgroundColor: '#000000',
          accentColor:     '#B3FE85',
          textColor:       '#FFFFFF',
          mutedColor:      '#6C757D',
        }
      }, '*'); // Use your actual origin in production: 'https://www.nexevo.io'

      console.warn('Could not post message to Dora iframe:', e);

```

Inside `dora-scene.html` (Dora's file), add this at the bottom of `<body>` to listen:

```html
<script>
  window.addEventListener('message', function(event) {
    // In production, verify: if (event.origin !== 'https://www.nexevo.io') return;
    if (event.data && event.data.type === 'NEXEVO_BRAND_TOKENS') {
      const tokens = event.data.payload;
      // Apply to Dora's root element
      document.documentElement.style.setProperty('--dora-primary', tokens.primaryColor);
      document.documentElement.style.setProperty('--dora-bg', tokens.backgroundColor);
      document.body.style.backgroundColor = tokens.backgroundColor;

  });
</script>
```

```
  X-Frame-Options: SAMEORIGIN
```
